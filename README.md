# online-education-project
本在线教育项目采用B2C模式的在线教育系统。
系统后台：讲师管理模块，课程分类管理模块，课程分类模块， 统计分析模块，订单管理。
系统前台：首页数据显示、讲师课程列表和详情、登录和注册、微信扫码 登录和支付
